# Portfolio VR Developer

Welcome to the Portfolio VR Developer project! This repository contains a professional portfolio website for a VR game developer, showcasing projects and providing a comprehensive CV.

## Features

- **Home Page**: Introduces the developer and links to other sections of the portfolio.
- **Projects Section**: Displays a list of projects with video uploads, allowing users to view and add new projects.
- **CV Section**: Presents the developer's professional experience, skills, and education.
- **Responsive Design**: The website is designed to be responsive and user-friendly across devices.

## Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/portfolio-vr-dev.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd portfolio-vr-dev
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```

5. **Open your browser** and go to `http://localhost:3000` to view the application.
